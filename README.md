#### 알고리즘 스터디 문서 작서 양식

- 양식에 맞게 `{}` 를 채우면 된다.

```html
<details>
 <summary>{문제이름 기입}</summary>
  <ul>
    <li>분류 : {분류 기입}</li>
    <li><a href="{문제 url 입력}">문제 보기</a></li>
    <!--if 문제 풀이가 애매해서 블로그를 참조했으면 -->
    <li><a href="{블로그 url 입력}">문제 보기</a></li>
    <!--else-->
    <li>{풀이과정 입력}</li>
  </ul>
</details>
```
